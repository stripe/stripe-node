# stripe-node (base branch)

**Internal use only.** This branch is hosted here for convenience but is intended
for use by Stripe-internal projects. It is not supported for general community
use and carries no stability guarantees for external consumers.

This branch contains the handwritten infrastructure of the stripe-node SDK —
everything that is not generated by sdk-codegen. It is used as a foundation by
internal code generation tools that produce typed SDKs for Stripe APIs.

## What's here

The SDK runtime: HTTP client, request/response handling, type coercion,
signature verification, pagination, error types, and configuration. Generated
resource files, tests, and documentation are excluded.

## Updating

This branch is rebuilt from `master` using the `create-node-base.sh` script in
the sdk-codegen toolchain. Do not commit directly to this branch — changes will
be overwritten on the next rebuild.
